Barn API


register_breed_definition(definition)
	definition the breeddefinition to use for breeding


breed definition:
{
	barn = "smallbarn" or "barn",
	food = list of food
			e.g. { "default:leaves" }
	mob1 = first or unisex mob required for breeding
	mob2 = second gender of mob required for breeding (optional)
	result = list of possible results e.g. male or female calfs
}
